# Assets — Interactivité ludique (TP Platformer)

L’ensemble des **ressources visuelles** utilisées dans le projet.

## Types d’assets présents
- Tilesets (sols, murs, environnements)
- Décors 2D
- Sprites de personnages
- Éléments d’ambiance (dungeon / platformer)

## Provenance des assets
Les assets utilisés dans ce projet proviennent principalement de **itch.io** :

- **Dark Dun** – par raou  
  https://raou.itch.io/dark-dun

- **Generic Dungeon Pack** – par bakudas  
  https://bakudas.itch.io/generic-dungeon-pack

- **Sunnyland – Fort of Illusion** – par ansimuz  
  https://ansimuz.itch.io/sunnyland-fort-of-illusion

## Créations originales
