# Scènes

Regroupe toutes les **scènes Godot (.tscn)** qui composent le jeu.

## Types de scènes possibles
Selon l’organisation du projet, ce dossier contient :
- Scène principale (niveau)
- Scènes de personnages (player, entités)
- Scènes d’environnement (plateformes, obstacles)
- Scènes utilitaires (UI, tests, prototypes)
